---
title: Unit Root
date: 10-31-2024
---

A unit root is a feature of some stochastic processes that disrupts the trend.
They can be particularly problematic in cases of statistical inference.

There are a number of tests that can be used to determine if a unit root exists
in the time series like the ADF test[^1].

[^1]: [Wikipedia.org - Unit Root](https://en.wikipedia.org/wiki/Unit_root)
[^2]: [Wikipedia.org - Augmented Dickey-Fuller test](https://en.wikipedia.org/wiki/Augmented_Dickey%E2%80%93Fuller_test)
